# 六法全書 学習支援アプリ

行政書士・司法書士・土地家屋調査士・宅建・公務員・司法試験に対応した六法全書学習Webアプリです。

## 機能

- **14法律・381条文**を収録（憲法・民法・刑法・商法・民訴・刑訴・行政手続法・行政不服申立法・行政事件訴訟法・国家賠償法・会社法・不動産登記法・土地家屋調査士法・司法書士法）
- 条文番号・キーワードによる全文検索
- 試験別頻出タグ絞り込み（司法・行政書士・司法書士・土地家屋調査士・宅建・公務員）
- 学習進捗トラッカー（理解度4段階）
- ランダム出題モード
- 条文メモ・ブックマーク
- データのエクスポート／インポート（スマホ↔PC同期）
- 未収録条文の手動追加

## 技術スタック

- React 18
- Vite
- Claude AI（設計・実装支援）

## 条文テキストについて

本アプリに収録されている法律条文は、日本国の法令（著作権法第13条により著作権の目的とならないもの）を出典としています。条文テキストの出典は [e-Gov 法令検索](https://laws.e-gov.go.jp) です。
