# 六法全書 学習支援アプリ

行政書士・司法書士・土地家屋調査士・宅建・公務員・司法試験に対応した六法全書学習Webアプリです。

## 機能

- **14法律・381条文**を収録（憲法・民法・刑法・商法・民訴・刑訴・行政手続法・行政不服申立法・行政事件訴訟法・国家賠償法・会社法・不動産登記法・土地家屋調査士法・司法書士法）
- 条文番号・キーワードによる全文検索
- 試験別頻出タグ絞り込み（司法・行政書士・司法書士・土地家屋調査士・宅建・公務員）
- 学習進捗トラッカー（理解度4段階）
- ランダム出題モード
- 条文メモ・ブックマーク
- データのエクスポート／インポート（スマホ↔PC同期）
- 未収録条文の手動追加

## 免責事項

本アプリの条文・解説は学習目的で作成されたものであり、正確性・最新性を保証するものではありません。法律上の判断が必要な場合は、必ず専門家にご相談ください。
